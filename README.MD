# Aetheris

Aetheris is a modern GPU-accelerated 2D simulation engine focused on real-time particle physics, interactive editing, AI-assisted simulation, and natural human interaction.

The project combines modern OpenGL compute shaders, a professional editor interface, and a modular architecture to build a scalable simulation platform.
